# redoc-test
Testing out OpenAPI spec generation using [Redoc](https://github.com/Redocly/redoc)

## How to run

1. Install [redoc-cli](https://redocly.com/docs/redoc/quickstart/cli/) globally using npm/x or yarn
2. Run `redoc-cli serve api-spec/user-api.yml --watch` from the root directory
3. Navigate to `http://127.0.0.1:8080`
